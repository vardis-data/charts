# lakehouse

Lakehouse serving layer — ClickHouse query engine and Cube semantic layer.

Requires the [Altinity clickhouse-operator](https://github.com/Altinity/clickhouse-operator) to be installed cluster-wide before deploying this chart.

```bash
helm install lakehouse oci://ghcr.io/vardis-data/charts/lakehouse --version 2.0.0
```
